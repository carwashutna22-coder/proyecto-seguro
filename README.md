# 🚗 Car Wash UTNA

Sistema de gestión para autolavado desarrollado como proyecto escolar.

## 📌 Descripción
Este sistema permite administrar empleados, servicios y clientes de un autolavado.

## 🛠️ Tecnologías
- Node.js
- Express
- MySQL
- HTML, CSS, JavaScript

## ⚙️ Instalación

1. Clonar repositorio:
git clone https://github.com/chico-gato21/car-wash-utna.git

2. Instalar dependencias:
npm install

3. Ejecutar proyecto:
npm start

## 🔐 Configuración

Crear archivo `.env` con:

DB_HOST=localhost  
DB_USER=root  
DB_PASSWORD=tu_password  
DB_NAME=carwash
